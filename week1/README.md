# workshop 4
- สร้างระบบตรวจสอบสิทธิ์เข้าใช้งานอย่างง่าย
- สร้างตัวแปรและกำหนดเงื่อนไข 
    1. ถ้า username , password และ age ตรงกับเงื่อนไขที่กำหนดให้แสดงผล "เข้าสู่ระบบสำเร็จ" โดยใช้ `&&` ในการเช็คเงื่อนไข
